# Forum

## 한국어

간단한 포럼입니다.

### 기능들

- 포스트 작성
- 포스트 수정
- 포스트 좋아요
- 포스트 삭제

### 기술 스택

- React
- Vite
- Redux Toolkit
- React Router DOM
- Tailwind CSS

### 실행하는 법

```bash
npm install
npm run dev
```
